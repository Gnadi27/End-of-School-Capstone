# End-of-School Capstone (German: HTL Diplomarbeit)

This repository contains the written thesis document of my end-of-school capstone project (German: *HTL Diplomarbeit*), along with the accompanying attachments that were originally provided as part of the submission.

**Project title:**  
*Automatisierte Be- und Entladung von Gasnitrierteilen*

The contents published here may differ slightly from the exact version that was officially submitted and graded.

## Overview

This project focuses on the design of an automated system for loading and unloading gas nitriding parts in an industrial heat-treatment environment.

The developed concept was built around an existing gas nitriding oven and targets the handling of two selector-shaft variants used in passenger-car gearboxes. On a high level, the work included the conception of the automated plant layout, the process flow between loading, buffering, charging, and unloading, as well as the development and selection of suitable handling, gripping, and manipulation approaches for the workpieces. The concept was validated on robotic motion simulations and thermo-static FEM. In addition, the project covered the integration of the main mechanical and automation-related subsystems into a coherent overall cell concept intended to reduce repetitive manual work, improve process reliability, reduce oven idle time, and increase productivity.
